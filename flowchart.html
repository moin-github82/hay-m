🔄 Data Flow — Key User Actions
Step-by-step for the most important flows

🔑 Login Flow

1. Enter email + password
2. POST /auth/login
3. bcrypt.compare(password)
4. Sign JWT (7d expiry)
5. Store token (session/async)
6. Navigate → Dashboard

🎯 Add Funds to Goal

1. Select goal + amount
2. Choose source (card / savings)
3. POST /goals/:id/funds
4. Deduct from card/savings
5. Add to goal.saved
6. Emit balance:updated

⚡ Round-up Save

1. User makes a purchase
2. App logs spend amount
3. POST /savings-tracker/roundup
4. ceil(spend) - spend = saved
5. Push to roundUps array
6. Emit savings:roundUp

💸 Send Money

1. Enter recipient email
2. Enter amount + note
3. POST /transactions/send
4. Debit sender card
5. Credit recipient card
6. Notify both users
